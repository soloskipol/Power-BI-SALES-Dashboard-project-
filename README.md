# Power-BI-SALES-Dashboard-project

## Project Overview
This Power BI dashboard provides an interactive analysis of sales data, helping stakeholders monitor performance across regions, products, and time periods.

## Tools Used
- Power BI
- DAX
- Excel

## Features
- Interactive filtering by product category
- Sales and profit KPIs
- Regional sales analysis
- Product performance tracking
- Time-based trend analysis


## Key Insights
- The West region generated the highest sales revenue.
- Technology products produced the highest profits.
- Sales performance varied significantly across regions.
- Interactive filters allow users to explore category-specific trends.
